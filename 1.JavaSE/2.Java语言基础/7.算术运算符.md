## +

* ##### 正数、加法、连接符

```java
class Demo03{
    public static void main(String[] args){
        int i = +10; // 正10
        System.out.println(i);

        int i1 = 1;
        int i2 = 2;
        System.out.println(i1+i2);
    }
}
```

* ### 连接符

  * ##### 作用 : 让任何数据都可以与字符串进行拼接 ;
  * ##### 注意 :

```java
1、+用于字符串的时候,其就是一个连接符(并不是做加法功能了)。
2、任何类型的数据与字符串使用连接符连接,结果都是字符串类型的数据。
```

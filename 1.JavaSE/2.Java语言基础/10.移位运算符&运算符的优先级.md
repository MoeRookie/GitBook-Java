* ##### 左移

```java
class Demo11{
    public static void main(String[] args){
        System.out.println(3<<1); // 6
    }
}
```

* ##### 计算过程 :

![](/assets/左移运算计算过程.png)

```java
如上图所示:
- 1、既然<<是位运算符,故将其变为其对应的二进制形式 - 00000011
- 2、左移1位就相当于整体向左移动1位,删除移出位并补全空位(空位补0) - 00000110(6)
- 3、左移2位 ... - 00001100(12)
```

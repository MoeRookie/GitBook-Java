```java
需求: 在控制台打印5句Hello,World!
- 本着写代码时应极力避免其重复的原则和重复代码实施繁杂程度的考虑,推荐使用循环语句来处理:
```

* ## 格式

```java
while(循环的条件){
    循环语句;
}
```

* ##### while循环语句的执行流程1 :

```java
/*
	需求:在控制台上打印5句Hello,World!
*/
class Demo04{
	public static void main(String[] args){
		while(true){
			System.out.println("Hello World!");
		}
	}
}
```

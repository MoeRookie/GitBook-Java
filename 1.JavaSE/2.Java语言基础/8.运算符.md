## 赋值运算符

### =

* ##### 赋值运算符

```java
class Demo06{
    public static void main(String[] args){
        int i = 10; // 将10赋值给i变量
        System.out.println("i="+i); // 10
    }
}
```

### +=

* ##### 加后赋值

```java
class Demo06{
	public static void main(String[] args){
		int i = 10; // 将10赋值给i变量
		i += 2; // i = i+2;
		System.out.println("i="+i); // 12
	}
}
```

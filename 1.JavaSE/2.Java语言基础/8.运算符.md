## 赋值运算符

### =

* ##### 赋值运算符

```java
class Demo06{
    public static void main(String[] args){
        int i = 10; // 将10赋值给i变量
        System.out.println("i="+i); // 10
    }
}
```

### +=

* ##### 加后赋值

```java
class Demo06{
    public static void main(String[] args){
        int i = 10; // 将10赋值给i变量
        i += 2; // i = i+2;
        System.out.println("i="+i); // 12
    }
}
```

* ##### 注意 : 对于此类赋值运算符 , Java编译器在编译的时候会自动进行强制类型转换\( 就不需要我们手动转换了 \);

```java
class Demo06{
    public static void main(String[] args){
        byte b1 = 1;
        byte b2 = 2;
        // b2 = b2 + b1; 报错:需要强制类型转换
        b2 += b1;
        System.out.println("b2="+b2); // 3
    }
}
```

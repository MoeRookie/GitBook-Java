## 常见问题

* #### NullPointerException - 空指针异常
* ##### 原因 : 引用类型变量没有指向任何对象 , 而访问了对象的属性或者是调用了对象的方法 .

```java
class Demo05{
	public static void main(String[] args){
		int[] niArr = new int[2];
		niArr = null; // 让该变量不要引用任何对象,不要记录任何内存地址.
		niArr[1] = 10;
		System.out.println(niArr[1]);
	}
}
```

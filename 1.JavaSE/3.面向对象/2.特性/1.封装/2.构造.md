## 构造方法

```java
需求 : 使用Java类描述一个婴儿.
分析:
    在现实生活中有两种婴儿,一种婴儿一出生就具备了名字(白户),还有一种婴儿是出生之后才有名字的(黑户).
    Java是面向对象的语言: 万物皆对象 - 任何事物都可以使用类进行描述.
/**
 *	需求 : 使用Java类描述一个婴儿.
 */
class Baby{
	int id; // 身份证号
 	String name; // 名字
	// 哭
	public void cry(){
		System.out.println(name + "哇哇的哭...");
	}
}
class Demo02{
	public static void main(String[] args){
		Baby baby = new Baby();
		baby.id = 110;
		baby.name = "狗娃";
		System.out.println("身份证号:"+baby.id+",名字:"+baby.name);
		baby.cry();
	}
}

```

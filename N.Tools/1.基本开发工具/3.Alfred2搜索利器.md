## 本地文件检索

```java
1. 使用Option+Space快捷键启动Alfred.
2. 右击"小帽子" -> Preferences -> Features
3. 列举了Alfred的一些基本功能(eg : 全文件检索 File Search)
```

* ##### open + 文件名 - 打开指定文件
* ##### find  + 文件名 - 打开指定文件所在目录
* ##### in + 搜索内容 - 搜索指定的文件内容
* ##### tags + 标志内容 - 根据指定的tag进行检索

## 思路

* ##### 使用自定义类封装数组
* ##### 添加类方法来实现数组操作

#### 自定义类

```java
package ch01;

public class MyArray {
    // 数组
    private long[] arr;
    // 数组中有效数据的大小
    private int elems;
    // 默认的构造方法
    public MyArray() {
        arr = new long[50];
    }
    public MyArray(int max) {
        arr = new long[max];
    }
    // 插入数据
    public void insert(long value) {
        arr[elems] = value;
        elems++;
    }
    // 显示数据
    public void display() {
        for(int i = 0;i < elems;i++) {
            System.out.print(arr[i] + " ");
        }
        System.out.println();
    }
    // 查找数据
    public int search(long searchKey) {
        int i;
        for(i = 0;i < elems;i++) {
            if(arr[i] == searchKey) {
                break;
            }
        }
        if(i == elems) {
            return -1;
        }else {
            return i;
        }
    }
    // 删除数据
    public void delete(long deleteKey) {
        if(search(deleteKey) == -1) {
            System.out.println("查找不到指定数据!");
        }else {
            for(int i = search(deleteKey);i < elems;i++) {
                arr[i] = arr[i+1];
            }
        }
    }
    // 修改数据
    public void update(long oldValue,long newValue) {
        if(search(oldValue) == -1) {
            System.out.println("查找不到指定数据!");
        }else {
            arr[search(oldValue)] = newValue;
        }
    }
}
```

#### 测试用例

```java
package ch01;

public class TestMyArray {
	public static void main(String[] args) {
		MyArray ma = new MyArray();
		// 插入数据
		ma.insert(20);
		ma.insert(545);
		ma.insert(3);
		ma.insert(545);
		ma.insert(6565);
		// 显示数据
		ma.display();
		// 查找数据
		System.out.println(ma.search(3)); // 2
		// 删除数据
		ma.delete(3);
		// 显示数据
		ma.display();
		// 修改数据
		ma.update(545, 778);
		// 显示数据
		ma.display();
	}
}
```
